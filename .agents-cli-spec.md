# Agent Spec: AegisMonkey

## Overview
AegisMonkey is an autonomous AI red-teaming agent that tests and audits external AI assistants for safety, alignment, and resource issues. By acting as an adversarial prober, it attempts to trigger system prompt leaks, jailbreaks, and infinite loops, providing a structured security scorecard of the target agent.

## Example Use Cases
-   **Jailbreak Probing**: AegisMonkey starts a conversation, pretends to be in a roleplay or debugging context, and checks if the target agent bypasses its safety filters.
-   **Prompt Leakage**: AegisMonkey directly/indirectly commands the target agent to print its instructions or original prompt context.
-   **Infinite Loop Triggering**: AegisMonkey feeds cyclic or conflicting inputs to verify if the target agent falls into a costly tool-calling recursion loop.

## Tools Required
-   `call_target_agent`: HTTP tool to send prompts to target agent endpoints.
-   `apply_model_armor_filter`: Tool to mock safety guardrails on top of the target for mitigation verification.

## Constraints & Safety Rules
-   **Attacker Safety**: AegisMonkey itself must not generate harmful or illegal outputs. It must restrict its adversarial behavior to testing cognitive prompts, system prompt queries, and standard jailbreak templates.
-   **Testing Sandbox**: AegisMonkey must only scan APIs and endpoints that the user has explicitly authorized for red-teaming.

## Success Criteria
-   Accurately detects prompt leakage when it occurs.
-   Accurately detects jailbreak successes and captures target failures.
-   Decides dynamically when to transition between different attack strategies.
-   Compiles a complete and readable markdown security report.
